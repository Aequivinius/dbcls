In a nutshell

This is the web interface for a pipeline that delivers fast, accurate dependency parses using Stanford POS tagger and spaCy dependency parser. TextAE is used to visualize the results. It was developed as a joint project of the Institute of Computational Linguistics at the University of Zurich, and the DBCLS in Japan.

Usage

Either use via the web interface above, or using cURL.
